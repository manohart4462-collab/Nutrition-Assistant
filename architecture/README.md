# Architecture:
The Nutrition Assistant project is designed using the MERN stack architecture to provide personalized nutrition recommendations and meal management.
The architecture of the system consists of multiple components including User Interface, Backend Services, Database, and External Services. These components work together to process user requests, store nutritional information, and generate personalized recommendations.

The architecture documents included in this folder are:
- Technical Architecture
- ER Diagram
- Key Features
- Roles and Responsibilities
- User Flow
- MVC Pattern
These documents provide a clear understanding of the system structure and data flow within the application.
