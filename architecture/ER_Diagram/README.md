# ER Diagram

The ER Diagram represents the database structure of the Nutrition Assistant application and the relationships between different entities.

The main entities in the system are:

- User
- Meal
- Food
- Diet Plan
- Nutrition Fact

A user can have multiple meals and diet plans. Meals contain nutritional information and are associated with food items. The ER Diagram helps in organizing data efficiently and maintaining relationships between collections in MongoDB.

The database design supports the implementation of the Nutrition Assistant application using the MERN stack.
