# User Flow

The following steps describe how a user interacts with the Nutrition Assistant application.

1. The user registers an account and completes their profile information.
2. The user logs into the application.
3. The user enters personal details such as age, weight, height, gender, and activity level.
4. The user searches for food items and records daily food consumption.
5. The application stores the food information and calculates nutritional values.
6. The user creates a diet plan based on personal goals.
7. The application provides nutritional information and meal suggestions.
8. The user monitors progress and views reports through the dashboard.
9. The user updates profile information and diet plans whenever required.

The user flow helps in understanding how users interact with different modules of the application from registration to progress tracking.
